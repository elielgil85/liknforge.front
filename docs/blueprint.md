# **App Name**: LinkForge

## Core Features:

- URL Input: Accept a long URL from the user via a text field.
- URL Shortening: Generate a shortened URL based on the input.
- Shortened URL Display: Display the shortened URL, a copy button, and a share button.
- Original URL Preview: Show a preview of the original, long URL to ensure transparency.
- Benefits Section: Provide short descriptions with minimalist icons highlighting the benefits of the URL shortener service.
- Footer Navigation: Include links to 'About Us', 'Contact', 'Terms of Service', and 'Privacy Policy' in the footer.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey trust and efficiency.
- Background color: Light gray (#F5F5F5), almost white, to ensure a clean and modern look.
- Accent color: Teal (#008080) for interactive elements to attract attention and signal interactivity.
- Body and headline font: 'Inter', sans-serif, for a clean and easily readable interface. Note: currently only Google Fonts are supported.
- Responsive layout optimized for mobile and desktop devices.
- Centralized input field and prominent action button for ease of use.
- Minimalist icons to represent the benefits of using the URL shortener.
